## Ciclo de vida

```
Proposed → Accepted → Deprecated → Superseded
```

Cada ADR comienza como **Proposed**, se mueve a **Accepted** tras aprobación, puede marcarse **Deprecated** si pierde validez, y debe referenciar un ADR **Superseded** que lo reemplaza.
